Task 2.1 Unit Converter App

About
This is a simple Android app made in Android Studio using Java for SIT305 Task 2.1.

The app lets users convert values in different categories:
- Currency
- Fuel Efficiency
- Temperature

Features
- Enter a value
- Choose a category
- Choose a unit to convert from
- Choose a unit to convert to
- Press convert to see the result

Tools Used
- Java
- XML
- Android Studio

Files
- `MainActivity.java` for the app logic
- `activity_main.xml` for the layout
- `strings.xml` for the spinner options
